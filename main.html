PPA Endpoint Information

GET /bmi

No required params

Will return all inputs and outputs stored in MongoDB in JSON format

POST /bmi

Required Params

KEY VALUE TYPE
feet Number
inches Number
weight Number

Will return the inputs and calculated outputs that have been stored in MongoDB in JSON format

GET /distance

No required params

Will return all inputs and outputs stored in MongoDB in JSON format

POST /distance

Required Params

KEY VALUE TYPE
x1 Number
y1 Number
x2 Number
y2 Number

Will return the inputs and calculated outputs that have been stored in MongoDB in JSON format
